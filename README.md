# RAGWatch AI Platform

A FastAPI-based RAG platform for uploading documents, generating embeddings, storing vectors, and answering questions.

## Technologies

- Python
- FastAPI
- LangChain
- FAISS
- Sentence Transformers

## Current Features

- Document processing
- Text chunking
- Embedding generation
- Vector storage

## Run the Application

pip install -r requirements.txt
uvicorn app.main:app --reload
